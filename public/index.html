Currency Converter

The project is a currency converter that allows users to convert values from Brazilian Real (BRL) to US Dollar (USD), British Pound (GBP) and Euro (EUR). The application uses updated exchange rates to perform conversions accurately and instantly provided from a API. Users can select the source currency (BRL) and immediately see the equivalent amounts in USD, GBP, and EUR which facilitate international financial transactions and value comparisons across different currencies in a practical and accessible manner. Created using JavaScript, HTML and CSS.
